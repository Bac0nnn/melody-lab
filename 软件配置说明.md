# 软件配置说明 — Melody Lab

## 一、项目概述

**Melody Lab** 是一个基于 Web Audio API 的交互式音乐创作与学习工具。支持两种运行方式：

| 方式 | 说明 | 推荐场景 |
|------|------|---------|
| 🆓 **零依赖版** | 直接双击 `index.html` 即可运行 | 本地体验、离线使用 |
| 🌐 **Next.js 版** | Node.js 开发服务器 + Vercel 部署 | 线上部署 |

## 二、零依赖版（推荐，无需安装）

**无需任何安装！** 直接：
1. 找到 `melody-lab/index.html` 
2. 双击打开
3. 浏览器中即可使用全部功能

**浏览器要求：** Chrome 105+ / Edge 105+ / Firefox / Safari（推荐 Chrome）

## 三、需要安装的软件（仅部署时需要）

| 软件 | 版本要求 | 用途 | 下载地址 |
|------|---------|------|---------|
| Node.js | v18+ | JavaScript 运行时 | https://nodejs.org/ |
| VS Code | 最新版 | 代码编辑器 | https://code.visualstudio.com/ |
| Git | v2+ | 版本控制 | https://git-scm.com/ |
| Vercel CLI | 最新版 | 部署上线 | `npm i -g vercel` |

## 四、安装与配置

### 1. 安装 Node.js
```bash
node --version   # 应显示 v18.x.x 或更高
npm --version    # 应显示 9.x.x 或更高
```

### 2. 安装 VS Code（可选）
推荐安装插件：ESLint、Prettier、GitLens

### 3. 配置 Git
```bash
git config --global user.name "你的名字"
git config --global user.email "你的邮箱"
```

### 4. 配置 GitHub 仓库
```bash
cd melody-lab
git remote add origin https://github.com/Bac0nnn/melody-lab.git
git push -u origin main
```

## 五、3 天提交计划

| 天数 | 提交内容 |
|------|---------|
| Day 1 | 项目初始化、钢琴键盘、和弦/音阶探索、音频可视化 |
| Day 2 | 钢琴扩展至 4 八度、键盘弹奏、录制回放、系统音频捕获 |
| Day 3 | QWERTY 映射、全宽自适应、音频转谱录制、预设示范曲、文档完善 |

## 六、开发模式
```bash
cd melody-lab
npm install      # 安装依赖
npm run dev      # 启动开发服务器 http://localhost:3000
```

## 七、部署到 Vercel
```bash
npm install -g vercel
vercel login
vercel --prod
```
或直接在 https://vercel.com 导入 GitHub 仓库自动部署。

## 八、零依赖 vs Next.js 版差异

| 功能 | 零依赖版 (index.html) | Next.js 版 |
|------|----------------------|-----------|
| 运行方式 | 双击浏览器打开 | `npm run dev` |
| 钢琴 | ✅ 4 八度全功能 | ✅ 2 八度 |
| 键盘弹奏 | ✅ QWERTY 映射 | ✅ 支持 |
| 录制回放 | ✅ 有 | ❌ |
| 音频可视化 | ✅ 3 种模式+音频捕获 | ✅ 3 种模式 |
| 和弦/音阶探索 | ✅ 有 | ✅ 有 |
| 音频转谱录制 | ✅ 有 | ❌ |
| 预设示范曲 | ✅ 有 | ❌ |
| REST API | ❌ | ✅ /api/chords 等 |
| Vercel 部署 | ❌ | ✅ |

> **建议：** 本地体验使用零依赖版，部署提交使用 Next.js 版。
